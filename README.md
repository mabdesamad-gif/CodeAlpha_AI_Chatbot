# 🤖 AI Chatbot — CodeAlpha Internship Task 4

An intelligent chatbot built with Flask and LLaMA 3.3 via Groq API.

## ✨ Features
- 🧠 Conversation memory
- 🌍 Multilingual (FR / AR / EN)
- 📂 Chat history saved in SQLite
- 🎨 Landing page with typing effect
- 📋 Copy button for responses

## 🛠️ Tech Stack
- Python / Flask
- Groq API + LLaMA 3.3
- HTML / CSS / JavaScript
- SQLite

## 🚀 Run locally
pip install -r requirements.txt
python app.py
